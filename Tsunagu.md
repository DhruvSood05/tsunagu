# TSUNAGU DESIGN SYSTEM v1.0

## Brand Philosophy

Tsunagu (繋ぐ) means "to connect" in Japanese.

Tsunagu is not another AI chatbot.

Tsunagu is an intelligent workspace that unifies:

- Gmail
- Google Calendar
- AI Assistance
- Scheduling
- Productivity Workflows

The design should communicate:

- Focus
- Precision
- Calmness
- Trust
- Intelligence
- Speed

Inspired by Japanese design principles:

- Simplicity
- Intentionality
- Negative Space
- Elegance
- Function over decoration

The experience should feel like a premium productivity tool used daily by founders, operators, executives, and power users.

---

# DESIGN PRINCIPLES

## 1. Focus First

Reduce visual noise.

Every component must serve a purpose.

Avoid decorative elements that do not improve usability.

---

## 2. Calm Interfaces

Use restrained color.

Avoid bright gradients and AI-style visual clutter.

The interface should feel peaceful even when displaying large amounts of information.

---

## 3. Premium Through Simplicity

The product should feel expensive because of:

- spacing
- typography
- hierarchy
- polish

Not because of flashy effects.

---

## 4. Keyboard First

Design every workflow with speed in mind.

The user should feel empowered.

Inspired by:

- Superhuman
- Raycast
- Linear
